RGB

A computer screen is made of pixels. A pixel is a tiny square made of three lights: one red, one blue, and one green.

Each pixel has an RGB code that says what color it should be.

What do you think RGB stands for?

1

Each light has a number from 255 (which means the light is all the way on) to 0 (all the way off).

A bright purple pixel has an RGB code of 255 000 255: the pixel's Red light is on, Green is off, and Blue is on.

170 000 170 makes a darker purple: the red and blue lights are a little dim, and green is off.

Hex

We use the decimal system to write numbers. Computers can save memory by using the hexadecimal system, or hex. It uses letters in place of some numbers.

0 00 10 0A 26 1A
1 01 11 0B 31 1F
2 02 12 0C 32 20
3 03 13 0D 33 21
4 04 14 0E 160 A0
5 05 15 0F 161 A1
6 06 16 10 175 AF
7 07 17 11 176 B0
8 08 18 12 240 F0
9 09 25 19 255 FF

In decimal white is 255255255, which is 9 numbers long. In hex white is ffffff, which is 6 numbers long. Because it takes up less space, computers often write colors in hex.

In CSS, you pick a font using the font-family property. font-family's value is a list of fonts. For example:

font-family: "Times New Roman", Times, serif;
font-family: cursive;
font-family: "Courier New", monospace;

If a font in font-family isn't installed on your computer, your computer tries the next one in the list. If instead of a font it's a family, like cursive, your computer uses any font in that family.

A font name that contains a must be inside "quotes".

Selectors

Every CSS rule starts with a selector. It tells the computer which parts of a website it should select to use with that CSS rule.

p, code, img{
  width: 400px;
}

This CSS rule says, Every paragraph, every code snippet, and every image should be 400 pixels wide.

p code img{
  width: 400px;
}

This CSS rule — without commas — says, Every image that's in a code snippet that's in a paragraph should be 400 pixels wide.

Stylesheets

All of a website's CSS is usually kept in a file called a stylesheet:

1..  p, code{
2..    color: #000000;
3..    font-size: 18px;
4..  }
5..  p{
6..    font-family: sans-serif;
7..  }
8..  code{
9..    font-family: monospace;
10.  }

Each of the chunks in {braces} is a rule. The p rule changes all the paragraphs on the website. The p, code rule changes all the paragraphs and code snippets. Rules add on to each other.

The C in CSS stands for cascading. This means three things:

  1. If a website has two stylesheets, it uses both of them.
  2. If a website has two CSS rules for paragraphs, it uses both rules.
  3. ...but if both rules have a font-family, the website will ignore the first font-family and use the second one instead.

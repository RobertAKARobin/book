# Fonts There are 5 //families// of fonts. Every font is in one of those families. Match each clue to its family. |\count num||count alpha| ||All letters are one space wide ||monospace ||Dutch for ''short line'' ||serif ||Begins with French for ''without'' ||sans-serif ||Fonts that look like handwriting||cursive ||Used mostly for logos and graphics ||fantasy |/
In CSS, you pick a font using the `font-family` property. For example: |\count num|code ||font-family: "Times New Roman", Times, serif; ||font-family: cursive; ||font-family: "Courier New", monospace; |/
If the first font in `font-family` isn't installed on your computer, your computer tries the next one. A font name that contains a _1_ must be inside `"quotes"`.

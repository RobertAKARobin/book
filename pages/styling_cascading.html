''Cascading'' means three things: -#If a website has two stylesheets, it uses both of them. - If a website has two CSS rules for paragraphs, it uses both rules. - ...**but** if both rules have a `font-family`, the website will ignore the first `font-family` and use the second one instead. -#/ >>===== ```# p{ background-color: #000000; color: #FFFF00; } code, p{ background-color: #00FF00; } code{ background-color: #000000; background-color: #0000FF; } ``/
This code means paragraphs will have _green_ backgrounds,
and code snippets will have _blue_ backgrounds. -=red - blue - yellow - green -=/
